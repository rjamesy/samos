## Notes
- Runtime refresh mechanism: routing captures a `SettingsSnapshot` at turn boundary (`SamOS/Domain/Routing/TurnPipeline.swift`) and applies it for deterministic route-planning decisions per turn.
- Dev-only controls are gated by `Developer mode`: Sam Gateway controls and plan-route override.
- Dynamic OpenAI model discovery is intentionally disabled in Phase 2. Model selection uses a local fallback preset list + freeform preferred model field.
